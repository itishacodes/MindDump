# ᨒ MindDump

**MindDump** is my personal digital garden and engineering portfolio—a networked brain space for hosting interactive project case studies, technical breakdowns, and custom design playgrounds.

---

## ⚡ Core Features

* **Instant SPA Transitions**: Client-side page swaps with zero-lag routing.
* **Interactive Graph**: A D3-powered node graph to visualize page connections.
* **Embedded Playgrounds**: Custom visual sandboxes and simulations compiled directly inside markdown files.
* **Frictionless Search**: Client-side index compiled for instant keyword search.

---

## 🛠️ Local Development

Start the local hot-reloading server in two steps:

```bash
# 1. Install dependencies
npm install

# 2. Start the dev server
npm run dev
```

Open **`http://localhost:8090`** in your browser to take it for a spin! 🚀
