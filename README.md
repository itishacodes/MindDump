## 🔮 What is MindDump?

Instead of posting generic updates on corporate feeds, **MindDump** is my networked brain garden. It is a custom digital garden system built on top of Preact, TypeScript, and D3.js, designed to feel like a high-end engineering dashboard rather than a static blog:

* ⚡ **Instant SPA Transitions**: Fast client-side page swaps powered by `micromorph` (no laggy reloads).
* 🎨 **Vibrant Visual Sandboxes**: Interactive inline playgrounds built into markdown posts (try the flood fill in Paint95 or the Web Audio drum machine in BitBox!).
* 🔍 **Instant Search Trie**: Client-side compiled index (`contentIndex.json`) for zero-latency keyword lookups.
* 🕸️ **Interactive Graph View**: Connected interactive D3 node graph lets you navigate concepts organically.

## 💻 Running MindDump Locally

Want to clone this digital garden and run it on your own machine? Fire up your terminal:

### 1. Clone the Repo
```bash
git clone https://github.com/itishacodes/MindDump.git
cd MindDump
```

### 2. Install Dependencies
```bash
npm install
```

### 3. Start the Local Server
This compiles the files and starts the local hot-reloading development server:
```bash
npm run dev
```

Open **`http://localhost:8090`** in your browser and take it for a spin! 🚀
