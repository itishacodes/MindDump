Did I write these down at 3 AM because I thought they would make me a millionaire? Yes.  
Will I look at them tomorrow and wonder what I was drinking? Probably.

Ideas are cheap, but they are also incredibly fun to collect. I’ve realized that my best product ideas hit me when I'm half-asleep, staring at a broken compiler loop. If I don't write them down immediately, they evaporate by the time I wake up.

Here is a live running list of startup ideas, side-project brainstorms, and cool web concepts floating around in my head right now.

---

### 💡 Startup Concepts

#### 1. DevCard API (Aesthetic Github Badges)
* **The Pitch**: A SaaS API that generates highly premium, vector-graphics credit cards for developer portfolios. Instead of standard flat README shields, it renders a glassy, animated credit card showing your GitHub stats, top languages, and peak coding hours.
* **Tech Stack**: Next.js, SVG generators, Canvas animation layers.
* **Vibe**: Glassmorphism, neon highlights, interactive hover tilt.

#### 2. LocalDoc RAG (Zero-Config Search)
* **The Pitch**: A drag-and-drop desktop utility that watches a local directory of PDFs and text files, automatically embeds them using a lightweight on-device model, and exposes a clean localhost JSON endpoint for search. No Docker setups, no complex Python environments—just a single binary.
* **Tech Stack**: Rust, Tauri, local embeddings database.
* **Vibe**: Minimalist, menubar app, lightning-fast.
